暫緩實施國小閩南語課綱

　

 

    最近，教育部研擬出新的國中小九年一貫閩南語課綱，要求小一學生能用閩南語與人輕鬆交談；小三能用羅馬拼音系統寫字並辨識各地方言差異，文言文或白話，而這套課綱將於兩年後實施。消息傳來，讓家長與教師嘩然。近年來我國中小學課程的擬定，由於缺乏專責機構負責，以致實施前僅能以臨時編組的形式委託辦理，實施的過程缺乏實驗與試辦，推動之後又忽略檢討、追蹤與評估的機制，課程就成了邊走邊改，邊做邊修的情況。前任杜正勝部長曾提出九七閩南語課綱草案，要求小一會用羅馬拼音認字，小三寫電子郵件，小五能在MSN上交談，國中生用閩南語寫部落格，當初這個關乎中小學語文課程的閩南語課綱，也就在引起社會喧然大波而暫緩。沒想到新政府上台之後，未見任何針對國中小實施以羅馬拼音來學習閩南語的可行性與必要性，進行檢討與研究，反而採取延後年級實施的妥協、和稀泥作法。不但看不出對於我國中小學生明確的語言政策，這樣的閩南語課綱只會加重中小學生學習閩南語的難度，也弱化了原已不足的國語教學。

 

   放眼國際上推動雙語教學的國家，無論是紐西蘭的英語和毛利話，或是加拿大的英語與法語，都重視特定優勢語言的保留，而不是強制對於兩種語言的同時學習，更遑論外加特殊的拼音系統。尤其對於小學階段，除非是特殊的種族學校，如毛利的語言巢外，都十分重視本國優勢語言(如英語)能力的培養，不但聽說讀寫並進，且視為培養國小學童建立核心能力的重要途徑。相較之下，台灣近年來國中小的國語文時數大幅刪減三分之一至一半，再加上考試標準化測驗的引導，過度強調修辭文法，因而在國際學童成就測驗中(如PISA, TIMSS)閱讀成績排行在東亞地區倒數的情況。針對上述這些問題，主管單位理應儘速全面檢討，而不是貿然將閩語拼音延緩實施年級等鋸箭法來應對。

 

    自民國86年實施建構式數學以來，不論是國中小的九年一貫課程或是高中的九五暫綱、九八課綱，都出現一些教改的盲點，例如：除了倉促研擬、匆忙上路，拼湊零碎、與政治妥協，尤其過程中大多是以委員會的任務編組方式，由教育部擬定政策、大學教授主持背書、中小學因應與配合，事前缺乏有效的實驗與評估，事後更不見追蹤評量與修正。這些問題不斷在教育界重演，若不及時全面檢討，其影響將會是動搖國本，恐怕下一代人的競爭力，在強勢語言不斷被弱化的情況之下，令人憂心忡忡。
